rust зҡ„ ownership жҳҜд»Җд№Ҳ

2018-11-01

еүҚиЁҖ

rust жҳҜ mozilla е…¬еҸёеҮәе“Ғзҡ„дёҖй—Ёзі»з»ҹзә§зј–зЁӢиҜӯиЁҖпјҢе…¶дё»иҰҒзү№зӮ№жҳҜжҖ§иғҪиүҜеҘҪ ( еҸ·з§°дёҚиҫ“ c++)пјҢиҝҗиЎҢе®үе…Ёд»ҘеҸҠж–ҮжЎЈдёҺе·Ҙе…·й“ҫе®Ңе–„гҖӮ rust еңЁеүҚз«Ҝж–№йқўд№ҹжңүеҫҲеӨҡеә”з”ЁпјҢе®ғжҳҜзӣ®еүҚ webassembly з”ҹжҖҒж”ҜжҢҒеәҰжңҖй«ҳзҡ„зј–зЁӢиҜӯиЁҖпјҢиҝҷдёӘеҸҜиғҪд№ҹи·ҹеҺҶеҸІеҺҹеӣ жңүе…іпјҢ webassembly зҡ„еүҚиә« asm.js д№ҹжҳҜ mozilla е…¬еҸёеҮәе“ҒпјҢжңүе…ҙи¶Јзҡ„еҗҢеӯҰеҸҜд»ҘиҮӘиЎҢжҹҘжүҫзӣёе…іиө„ж–ҷгҖӮжӯӨеӨ–пјҢеӨ§еҗҚйјҺйјҺзҡ„ deno е…¶еә•еұӮе®һзҺ°д№ҹжҳҜеҹәдәҺ rust зј–еҶҷгҖӮжң¬ж–Үдё»иҰҒд»Ӣз»Қд»ӨеҫҲеӨҡеҲқеӯҰиҖ…ж„ҹеҲ°жҮөеңҲпјҢеҜ№зј–еҶҷ rust д»Јз Ғдә§з”ҹдәәз”ҹжҖҖз–‘зҡ„жқҖжүӢ -ownership

Ownership жҳҜд»Җд№Ҳ

OwnershipпјҢдёӯж–Үзҝ»иҜ‘жҳҜжүҖжңүжқғпјҢе®ғжҳҜ rust зЁӢеәҸзҡ„дёҖеҘ—еҶ…еӯҳз®ЎзҗҶзӯ–з•ҘгҖӮеҜ№дәҺзј–зЁӢиҜӯиЁҖжқҘиҜҙпјҢеҶ…еӯҳз®ЎзҗҶжҳҜдёҖдёӘж°ёжҒ’зҡ„иҜқйўҳпјҢжңүзҡ„зј–зЁӢиҜӯиЁҖиҮӘеёҰеһғеңҫеӣһж”¶пјҲ GCпјүпјҢ GC дјҡиҙҹиҙЈиҝҪиёӘе’Ңжё…йҷӨдёҚеҶҚдҪҝз”Ёзҡ„еҶ…еӯҳеҸҳйҮҸпјҢжҜ”еҰӮжҲ‘д»¬еёёеҶҷзҡ„ jsгҖӮеҜ№дәҺжІЎжңү GC зҡ„зј–зЁӢиҜӯиЁҖжқҘиҜҙпјҢе°ұйңҖиҰҒејҖеҸ‘иҖ…еңЁзј–з ҒдёӯиҮӘиЎҢзЎ®е®ҡе“ӘдәӣеҸҳйҮҸдёҚеҶҚдҪҝз”Ёе№¶жүӢеҠЁжё…йҷӨе®ғпјҢиҝҷеҜ№дәҺејҖеҸ‘иҖ…жқҘиҜҙжҳҫз„¶жҳҜдёҖдёӘжҜ”иҫғеӨ§зҡ„иҙҹжӢ…пјҢеҰӮжһңж“ҚдҪңдёҚеҪ“жһҒжҳ“еҜјиҮҙеҶ…еӯҳжі„йңІжҲ–иҖ…з©әжҢҮй’Ҳд№Ӣзұ»зҡ„ејӮеёёпјҢе…ёеһӢзҡ„жҜ”еҰӮ c/c++гҖӮеңЁеҶ…еӯҳз®ЎзҗҶзӯ–з•ҘдёҠ rust еұһдәҺеҸҰиҫҹи№Ҡеҫ„пјҢе®ғдјҡеңЁзј–иҜ‘йҳ¶ж®өжҢүз…§ ownership и§„еҲҷжЈҖжҹҘдҪ зЁӢеәҸдёӯзҡ„еҶ…еӯҳйҡҗжӮЈпјҢд»ҺиҖҢдҝқиҜҒзЁӢеәҸеңЁиҝҗиЎҢж—¶зҡ„е®үе…ЁзЁіе®ҡгҖӮ ownership и§„еҲҷеҸҜд»Ҙз®ҖзәҰзҡ„жҰӮжӢ¬дёәд»ҘдёӢдёүзӮ№пјҡ

  1. еңЁ rust зЁӢеәҸдёӯжҜҸдёҖдёӘиў«иөӢеҖјзҡ„еҸҳйҮҸжҳҜиҜҘеҖјзҡ„жүҖжңүиҖ…
  2. еҜ№дәҺдёҖдёӘеҖјжқҘиҜҙпјҢеҗҢдёҖж—¶й—ҙеҸӘиғҪеӯҳеңЁдёҖдёӘжүҖжңүиҖ…
  3. еҪ“жүҖжңүиҖ…жүҖеұһдҪңз”Ёеҹҹз»“жқҹж—¶пјҢжүҖжңүиҖ…зҡ„еҖјд№ҹдјҡиў«й”ҖжҜҒ

иҝҷдёүзӮ№е…ҲдёҚеҒҡж·ұе…Ҙд»Ӣз»ҚпјҢеҸҜд»Ҙе…ҲеңЁи„‘еӯҗйҮҢз•ҷдёӘеҚ°иұЎпјҢеҗҺз»ӯдјҡи®ІеҲ°гҖӮзӣ®еүҚжҲ‘д»¬йңҖиҰҒзҹҘйҒ“зҡ„жҳҜдёәд»Җд№Ҳ rust дјҡжңү ownership жңәеҲ¶пјҢе®ғе…·дҪ“и§ЈеҶід»Җд№Ҳй—®йўҳгҖӮ

Ownership и§ЈеҶід»Җд№Ҳй—®йўҳ

ownership дё»иҰҒи§ЈеҶізҡ„жҳҜе ҶдёҠж•°жҚ®еҶ…еӯҳзҡ„з®ЎзҗҶй—®йўҳгҖӮзӣёдҝЎжңүиҝҮзј–зЁӢз»ҸйӘҢзҡ„еҗҢеӯҰйғҪзҹҘйҒ“пјҢеҹәжң¬дёҠдёҚи®әжҳҜејәзұ»еһӢиҝҳжҳҜејұзұ»еһӢзј–зЁӢиҜӯиЁҖйғҪеӯҳеңЁз®ҖеҚ•ж•°жҚ®зұ»еһӢпјҢиҜёеҰӮ stringпјҢ intпјҢ float,boolean е’ҢеӨҚжқӮж•°жҚ®зұ»еһӢпјҢиҜёеҰӮ structпјҢ object зӯүзӯүгҖӮдёҚеҗҢзҡ„ж•°жҚ®зұ»еһӢеҚ з”Ёзҡ„еҶ…еӯҳеҢәеҹҹжҳҜдёҚеҗҢзҡ„гҖӮеҜ№дәҺз®ҖеҚ•ж•°жҚ®зұ»еһӢжқҘиҜҙпјҢеңЁејәзұ»еһӢиҜӯиЁҖдёӯпјҢе®ғд»¬еңЁеЈ°жҳҺзҡ„ж—¶еҖҷеҸҜдҪҝз”Ёзҡ„еҶ…еӯҳеӨ§е°ҸжҳҜеӣәе®ҡе·ІзҹҘзҡ„пјҢиҝҷйғЁеҲҶж•°жҚ®еӯҳж”ҫеңЁеҶ…еӯҳдёӯзҡ„ж ҲпјҲ stackпјүеҢәгҖӮеҜ№дәҺеӨҚжқӮж•°жҚ®зұ»еһӢжқҘиҜҙпјҢз”ұдәҺе…¶ж•°жҚ®еҖјдјҡеҠЁжҖҒеҸҳеҢ–пјҢеӣ жӯӨе…¶жүҖеҚ еҶ…еӯҳеӨ§е°Ҹж— жі•еңЁзј–иҜ‘йҳ¶ж®өзЎ®е®ҡпјҢе…¶еңЁеҶ…еӯҳдёӯеӯҳеӮЁж—¶дҪҝз”Ёзҡ„жҳҜе ҶпјҲ heapпјүеҢәгҖӮдёҖдёӘе…ёеһӢеңәжҷҜжҳҜеңЁејҖеҸ‘е‘Ҫд»ӨиЎҢеә”з”Ёж—¶пјҢдҪ йңҖиҰҒе°Ҷз”ЁжҲ·зҡ„иҫ“е…ҘиөӢеҖјз»ҷдёҖдёӘеҸҳйҮҸпјҢеӣ дёәз”ЁжҲ·иҫ“е…ҘжҳҜдёҚеҸҜйў„зҹҘзҡ„пјҢжүҖд»Ҙд№ҹеҸӘиғҪеңЁиҝҗиЎҢж—¶жүҚиғҪзЎ®е®ҡиҜҘеҸҳйҮҸйңҖиҰҒзҡ„еҶ…еӯҳз©әй—ҙеӨ§е°ҸгҖӮ

дёҚи®әжҳҜз®ҖеҚ•зұ»еһӢиҝҳжҳҜеӨҚжқӮзұ»еһӢпјҢеҪ“е…¶жүҖеңЁдҪңз”Ёеҹҹз»“жқҹж—¶ rust еҶ…йғЁдјҡиҮӘеҠЁи°ғз”Ё drop ж–№жі•жё…йҷӨе…¶жүҖеҚ еҶ…еӯҳпјҢдҪҶй—®йўҳжҳҜеҰӮжһңжңүеӨҡдёӘеҸҳйҮҸеҗҢж—¶жҢҮеҗ‘еҗҢдёҖеқ—еҶ…еӯҳеҢәеҹҹж—¶еҰӮжһңиҝҷдәӣеҸҳйҮҸйғҪиў«й”ҖжҜҒпјҢдјҡеҜјиҮҙеҗҢдёҖеқ—еҶ…еӯҳеҢәеҹҹиў«йҮҠж”ҫеӨҡж¬ЎпјҢиҝҷеҗҢж ·дјҡеҜјиҮҙеҶ…еӯҳеҙ©жәғпјҢеёҰжқҘе®үе…ЁйҡҗжӮЈгҖӮиҖҢ ownership е°ұжҳҜ rust еҜ№дәҺиҝҷдёӘй—®йўҳзҡ„и§Јжі•пјҢйӮЈд№Ҳе®ғжҳҜеҰӮдҪ•еңЁд»Јз ҒеұӮйқўжқҘдҪ“зҺ°зҡ„е‘ўпјҹдё”зңӢеҰӮдёӢеҲҶи§Ј

Ownership зҡ„д»Јз ҒдҪ“зҺ°

йҰ–е…ҲжқҘзңӢдёҖз»„д»Јз ҒдёҺжү§иЎҢз»“жһңжј”зӨәпјҡ

  • з®ҖеҚ•зұ»еһӢпјҡ

    жү§иЎҢз»“жһңпјҡ
  • еӨҚжқӮзұ»еһӢпјҡ

    жү§иЎҢз»“жһңпјҡ

дёҠйқўдёӨз»„д»Јз Ғзҡ„жү§иЎҢйҖ»иҫ‘еҹәжң¬дёҖиҮҙпјҢдҪҶжҳҜжүҖеҫ—з»“жһңеҚҙе®Ңе…ЁдёҚеҗҢгҖӮеңЁзӨәдҫӢдёҖдёӯпјҢ x еҸҳйҮҸзҡ„еҖјз»‘е®ҡеҲ° y еҗҺпјҢжү“еҚ° x е’Ң y зҡ„еҖјеҸҜд»ҘжӯЈеёёжҳҫзӨәжғіиҰҒзҡ„з»“жһңгҖӮдҪҶеңЁзӨәдҫӢдәҢдёӯпјҢ s1 еҖјз»‘е®ҡеҲ° s2 еҗҺпјҢеҪ“жү“еҚ° s1 е’Ң s2 еҖјж—¶ rust еҚҙеңЁзј–иҜ‘йҳ¶ж®өе°ұзӣҙжҺҘжҠҘй”ҷдәҶпјҢиҝҷжҳҜдёәд»Җд№Ҳе‘ўпјҹгҖӮз®ҖеҚ•зұ»еһӢдёҺеӨҚжқӮзұ»еһӢеңЁйҖҡиҝҮеҸҳйҮҸиөӢеҖјж—¶зҡ„еҶ…йғЁйҖ»иҫ‘йҡҫйҒ“жңүжүҖе·®ејӮпјҹжІЎй”ҷпјҢзЎ®е®һжҳҜеӯҳеңЁе·ЁеӨ§е·®ејӮпјҢжҲ‘дјҡз”ЁеҮ еј зӨәж„ҸеӣҫжқҘиҝӣиЎҢжј”зӨәиҜҙжҳҺгҖӮ
еңЁзӨәдҫӢдёҖз§Қеӣ дёәеҸҳйҮҸ x жҳҜз®ҖеҚ•зұ»еһӢпјҢеӣ жӯӨе…¶еҖјжҳҜеӯҳж”ҫеңЁж ҲдёҠзҡ„пјҢеҪ“иөӢеҖјз»ҷеҸҳйҮҸ y ж—¶жү§иЎҢзҡ„е…¶е®һжҳҜеҖјжӢ·иҙқзҡ„иҝҮзЁӢпјҢе…¶з»“жһңжҳҜдјҡеңЁж ҲеҶ…еӯҳдёҠж–°е»әдёҖдёӘз©әй—ҙеӯҳж”ҫ y зҡ„еҖјпјҢзӨәж„ҸеӣҫеҰӮдёӢпјҡ

еӣ дёә x е’Ң y еҲҶеҲ«жҢҮеҗ‘дёҚеҗҢзҡ„еҶ…еӯҳеҢәеҹҹпјҢеңЁ main еҮҪж•°з»“жқҹйҮҠж”ҫеҶ…еӯҳж—¶д№ҹе°ұдёҚеӯҳеңЁеҶ…еӯҳжүҖжңүжқғеҶІзӘҒй—®йўҳпјҢеӣ жӯӨеңЁ main еҮҪж•°жң«е°ҫжү“еҚ° x е’Ң y еҸҳйҮҸзҡ„еҖјеҸҜд»ҘжӯЈеёёеұ•зӨәгҖӮ

еңЁзӨәдҫӢдәҢдёӯ String жҳҜеӨҚжқӮзұ»еһӢпјҲжӯӨеӨ„дёҚиҰҒеёҰе…Ҙ js зҡ„жҰӮеҝөпјҢеңЁ rust дёӯ String з”ЁдәҺеӯҳеӮЁеҠЁжҖҒеҸҳеҢ–зҡ„еӯ—з¬ҰдёІпјҢеұһдәҺеӨҚжқӮзұ»еһӢпјүпјҢеҸҳйҮҸ s1 зҡ„еҖјеӨҚеҲ¶еҲ° s2 ж—¶жү§иЎҢзҡ„е…¶е®һжҳҜжҢҮй’Ҳзҡ„еӨҚеҲ¶пјҢзӨәж„ҸеӣҫеҰӮдёӢпјҡ

д»Һ s1 еҲ° s2 пјҢе…¶жҢҮй’ҲжҢҮеҗ‘зҡ„йғҪжҳҜе ҶеҢәзҡ„еҗҢдёҖдёӘеҶ…еӯҳз©әй—ҙпјҢеңЁиҝҷз§Қжғ…еҶөдёӢеҪ“ s1 е’Ң s2 жүҖеңЁдҪңз”Ёеҹҹз»“жқҹйңҖиҰҒй”ҖжҜҒеҸҳйҮҸйҮҠж”ҫеҶ…еӯҳж—¶еҰӮжһңдёҚеҠ еӨ„зҗҶе°ұдјҡеҸ‘з”ҹеҗҢдёҖеқ—еҶ…еӯҳеҢәеҹҹиў«йҮҠж”ҫеӨҡж¬Ўзҡ„й—®йўҳпјҢдёәдәҶйҒҝе…ҚиҝҷдёӘй—®йўҳпјҢ rust йҮҮз”ЁдәҶ ownership жңәеҲ¶пјҢеӣһйЎҫеүҚйқўжҲ‘д»¬д»Ӣз»Қ ownership ж—¶жҸҗеҲ°зҡ„дёүзӮ№жҰӮжӢ¬пјҢеҜ№дәҺдёҖдёӘеҖјжқҘиҜҙпјҢеҗҢдёҖж—¶й—ҙеҸӘиғҪеӯҳеңЁдёҖдёӘжүҖжңүиҖ…гҖӮеҪ“ s1 иөӢеҖјеҲ° s2 ж—¶еӣ дёәиҝҷдёӨдёӘжүҖжңүиҖ…жҢҮеҗ‘зҡ„жҳҜеҗҢдёҖдёӘеҶ…еӯҳз©әй—ҙпјҢеӣ жӯӨеңЁиөӢеҖје®ҢжҲҗеҗҺ s1 дјҡиў«й”ҖжҜҒпјҢ s2 жҺҘиҝҮиҜҘеҶ…еӯҳз©әй—ҙзҡ„жүҖжңүжқғпјҢиҝҷж ·еҜјиҮҙзҡ„з»“жһңе°ұжҳҜжҲ‘д»¬еңЁзӨәдҫӢдәҢзЁӢеәҸзҡ„жү§иЎҢз»“жһңдёӯзңӢеҲ°зҡ„йӮЈж ·пјҢеҸҳйҮҸ s1 дёҚиғҪз»§з»ӯи®ҝй—®гҖӮзӨәж„ҸеӣҫеҰӮдёӢпјҡ

йҷӨдәҶжҳҫејҸзҡ„еҸҳйҮҸиөӢеҖјд»ҘеӨ–пјҢеёёи§Ғзҡ„еҮҪж•°еҸӮж•°дј йҖ’д№ҹжҳҜеӨ§еһӢ ownership дәӨжҺҘзҺ°еңәгҖӮзӨәдҫӢд»Јз ҒеҰӮдёӢжүҖзӨәпјҡ

йЎәйҒ“еҶҚеӨҡжүҜдёҖеҸҘпјҢиҝҷз§ҚжҢҮй’Ҳзҡ„еӨҚеҲ¶дёҺ js дёӯеӨҚжқӮзұ»еһӢзҡ„еј•з”Ёдј йҖ’жҜ”иҫғзұ»дјјпјҢдҪҶ js дёӯеј•з”Ёдј йҖ’ж—¶е№¶дёҚдјҡеғҸ rust иҝҷж ·й”ҖжҜҒ s1, иҖҢжҳҜ s1,s2 еҗҢж—¶еӯҳеңЁпјҢдё”дёҖж—Ұ s2 зҡ„еҖјжңүжүҖеҸҳжӣҙпјҢ s1 зҡ„еҖјд№ҹдјҡжӣҙж–°пјҢиҝҷд№ҹз»ҷ js ејҖеҸ‘иҖ…еңЁиҝҗиЎҢж—¶иҝҪиёӘж•°жҚ®еҸҳжӣҙзӣёе…ізҡ„ bug еўһеҠ дәҶеӣ°йҡҫпјҢиҝҷд№ҹжҳҜдёәд»Җд№Ҳ js дјҡжңүеҗ„з§Қеҗ„ж ·зҡ„ immutable еә“гҖӮ
